# 🚀 FlowNote v2.0.0

**FlowNote** adalah aplikasi manajemen produktivitas dan keuangan *all-in-one* yang dirancang dengan estetika premium dan pengalaman pengguna yang mulus. Kelola pengeluaran, simpan catatan penting, dan pantau analisis finansial Anda dalam satu genggaman.

## ✨ Fitur Utama

### 💰 Manajemen Keuangan (Finance)
- **Pelacakan Saldo**: Lihat saldo total, pemasukan, dan pengeluaran bulan ini dalam sekejap.
- **Riwayat Transaksi**: Daftar transaksi yang rapi dengan kategori berwarna dan ikon ikonik.
- **Pill Progress Bar**: Pantau tingkat tabungan (*savings rate*) Anda secara visual.

### 📝 Catatan & Tugas (Notes & Tasks)
- **Grid & List View**: Pilih tampilan yang paling nyaman (kotak atau daftar).
- **Warna Kustom**: Kelompokkan catatan Anda berdasarkan kategori warna yang cantik.
- **Sistem Tugas**: Tandai tugas yang sudah selesai dengan animasi checklist yang memuaskan.

### 📊 Analitik Premium (Analytics)
- **Dashboard Visual**: Grafik batang dan lingkaran (*pie chart*) yang interaktif.
- **Ambient Glow Effects**: Desain mutakhir dengan efek blur dan gradasi yang memberikan kesan mewah.
- **Tren 6 Bulan**: Pantau riwayat keuangan Anda dalam jangka panjang.

### 🌓 Dark Mode & Kustomisasi
- Dukungan penuh untuk mode gelap yang nyaman di mata.
- **Pill Navigation Bar**: Navigasi minimalis yang responsif dan modern.

## 🛠️ Tech Stack & Arsitektur

Aplikasi ini dibangun menggunakan teknologi terkini untuk memastikan performa maksimal:

- **Frontend**: Flutter (Dart)
- **State Management**: Riverpod (StateNotifier)
- **Database & Auth**: Firebase (Authentication & Cloud Firestore)
- **Navigasi**: GoRouter
- **Grafik**: FlChart
- **Animasi**: Animate Do & Custom Tweens

## 🚀 Instalasi

1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/grsll/flownote_app.git
   ```

2. **Dapatkan paket dependensi:**
   ```bash
   flutter pub get
   ```

3. **Konfigurasi Firebase:**
   - Tambahkan file `google-services.json` ke folder `android/app/`.
   - Tambahkan akun Firebase untuk iOS/Web jika diperlukan.

4. **Jalankan Aplikasi:**
   ```bash
   flutter run
   ```
